PDF Nova Desktop - Installation Windows

1) Extraire ce dossier.
2) Clic droit sur install.cmd -> "Executer en tant qu'administrateur" (recommande) ou double-clic.
3) L'application sera installee ici:
   %LOCALAPPDATA%\Programs\PDFNovaDesktop
4) Raccourcis crees:
   - Bureau: PDF Nova Desktop
   - Menu Demarrer: PDF Nova Desktop

Desinstallation:
- Lance %LOCALAPPDATA%\Programs\PDFNovaDesktop\uninstall.cmd
